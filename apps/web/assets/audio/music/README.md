# Background Music Assets

These tracks are legal style alternatives for an 80s dark synth / supernatural
game atmosphere. They are not music from *Stranger Things* and must not be
renamed or presented as official soundtrack material.

## Files

| Local file | Original title | Author | License | Suggested use |
|---|---|---|---|---|
| `retro-synth-main.mp3` | Stranger Things Inspired Synth | yd | CC0 1.0 | Main menu or opening |
| `darker-waves.mp3` | Darker Waves | Zander Noriega | CC BY 3.0 | Dark menu or result screen |
| `synthwave-horror-loop.ogg` | Synthwave Horror Loop | remaxim | CC BY 4.0 | Short gameplay ambience loop |
| `retro-synth-drive.ogg` | Slampe Retro Synth | yd | CC0 1.0 | Faster, energetic gameplay |

`synthwave-horror-loop.m4a` and `retro-synth-drive.m4a` are local AAC
transcodes of the corresponding OGG source files for Safari/iPhone playback.
They retain the same authors and licenses.
