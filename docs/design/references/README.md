# Visual References

These files are protected design references for the opening warning sequence.
They are documentation assets and are not loaded by the browser at runtime.

- `medical-warning-copy.png`: approved Chinese copy and interaction concept
- `medical-warning-layout.png`: black background, red alert panel, yellow warning
  triangle, and centered hierarchy

The implementation lives in:

- `apps/web/index.html`
- `apps/web/src/bootstrap/intro-animation.js`
- `apps/web/src/main.js`
